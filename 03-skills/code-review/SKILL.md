# Code Review Skill / 代码审查技能

这个 skill 用于做结构化代码审查，重点关注：

1. **Security Analysis**
2. **Performance Review**
3. **Code Quality**
4. **Maintainability**

## Review Template

### Summary

- Overall quality assessment
- Key findings count
- Recommended priority areas

### Critical Issues

- **Issue**
- **Location**
- **Impact**
- **Severity**
- **Fix**

### Findings by Category

#### Security

列出安全漏洞或风险点

#### Performance

列出性能问题与复杂度风险

#### Quality

列出命名、结构、文档和测试问题

#### Maintainability

列出可维护性问题和重构建议
