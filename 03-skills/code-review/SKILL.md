# 代码审查技能（Code Review Skill）

这个 skill 用于做结构化代码审查，重点关注：

1. **Security Analysis**
2. **Performance Review**
3. **Code Quality**
4. **Maintainability**

## 审查模板

### 总结

- Overall quality assessment
- Key findings count
- Recommended priority areas

### 严重问题

- **Issue**
- **Location**
- **Impact**
- **Severity**
- **Fix**

### 按类别列出问题

#### 安全

列出安全漏洞或风险点

#### 性能

列出性能问题与复杂度风险

#### 质量

列出命名、结构、文档和测试问题

#### 可维护性

列出可维护性问题和重构建议
