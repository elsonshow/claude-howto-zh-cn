# 品牌语气技能

## 概览

这个 skill 用于确保所有对外表达保持统一的 brand voice、tone 和 messaging。

## 品牌身份

### 使命

帮助团队用 AI 自动化开发工作流

### 价值观

- **Simplicity**
- **Reliability**
- **Empowerment**

### 语气风格

- 友好但专业
- 清晰简洁
- 自信但不过度夸张
- 能理解用户痛点

## 写作指南

### 推荐写法

- 面向读者说“你”
- 多用主动语态
- 开头先讲价值
- 多给具体例子
- 句子尽量简洁

### 避免写法

- 避免企业黑话
- 避免居高临下
- 避免空洞夸张词
- 避免大段无层次文本

## 词汇

### 推荐术语

- Claude
- Code generation
- Agent
- Streamline
- Integrate

### 避免术语

- Cutting-edge
- Game-changer
- Leverage
- Utilize
- Paradigm shift
