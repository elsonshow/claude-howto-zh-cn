# API Documentation Generator Skill / API 文档技能

## Generates

- OpenAPI / Swagger 文档
- endpoint 文档
- SDK 示例
- 集成指南
- 错误码说明
- 鉴权说明

## For Each Endpoint

输出尽量包含：

- Description
- Parameters
- Response
- Errors
- cURL / JavaScript / Python 示例

要求：

- 文档结构清晰
- 示例可复制
- 与源码保持一致
