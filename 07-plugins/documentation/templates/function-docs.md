# 函数文档模板：`functionName`

## 说明

简要说明函数做什么。

## 函数签名

```typescript
function functionName(param1: Type1, param2: Type2): ReturnType
```

## 参数

| 参数 | 类型 | 是否必填 | 说明 |
|-----------|------|----------|-------------|
| param1 | Type1 | Yes | param1 说明 |
| param2 | Type2 | No | param2 说明 |

## 返回值

**类型**: `ReturnType`

## 可能抛出的错误

- `Error`
- `TypeError`

## 示例

```typescript
const result = functionName('value1', 'value2');
console.log(result);
```
