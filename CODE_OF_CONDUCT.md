# 贡献者行为准则

## 我们的承诺

我们希望把这个项目维护成一个友好、尊重、多元、可持续合作的社区。

## 我们鼓励的行为

- 尊重他人
- 乐于协作
- 接受建设性反馈
- 欢迎新贡献者
- 讨论问题时聚焦内容而不是攻击个人

## 不可接受的行为

- 骚扰、歧视、侮辱
- 人身攻击
- 威胁、恐吓
- 恶意刷屏或故意破坏讨论
- 因他人举报不当行为而报复

## 适用范围

适用于：

- GitHub issues
- pull requests
- discussions
- 其他与本项目相关的公开或私下沟通

## 处理原则

维护者会根据情况采取合理措施，包括提醒、警告、限制参与或移除权限。
